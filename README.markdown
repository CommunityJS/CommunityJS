# CommunityJS.org
-----------
This is the software behind CommunityJS.org


### Quick start
-----------

Clone the repo

```
$ git clone git://github.com/CommunityJS/CommunityJS.git
```

Run server

```
$ node server.js
```

Check your browser

```
http://0.0.0.0:8000/
```
